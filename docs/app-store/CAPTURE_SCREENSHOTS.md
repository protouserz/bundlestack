# App Store screenshots

Shopify recommends **1600×900** PNG or JPEG. Capture these from your dev store after logging into admin.

## Required shots (admin — log in first)

Open: `https://admin.shopify.com/store/bundlestack-dev/apps/bundlestack`

| # | File | How to capture |
|---|------|----------------|
| 1 | `01-dashboard.png` | BundleStack dashboard with store health stats |
| 2 | `02-create-offer.png` | Offers → Create offer (product picker + tiers) |
| 3 | `03-billing.png` | Billing page showing free tier + pricing plans |
| 4 | `04-storefront-widget.png` | Product page with BundleStack theme block visible |
| 5 | `05-checkout-discount.png` | Checkout with quantity discount applied (optional) |

### macOS shortcut

`Cmd + Shift + 4` → drag to select window → saves to Desktop.

Resize browser window to ~1600×900 before capturing (Chrome DevTools → Toggle device toolbar → Responsive → 1600×900).
